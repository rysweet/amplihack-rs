# Glossary

- **crate** — a Rust compilation unit (library or binary) in the workspace.
- **bin** — a binary target (`amplihack`, `amplihack-hooks-bin`, `amplihack-asset-resolver-bin`).
- **hooks** — PreToolUse/PostToolUse handlers that gate/augment launcher tool calls.
- **recipe runner** — executes YAML workflow recipes as agent + shell steps.
- **amplifier-bundle** — bundled agents/skills/context staged into `~/.copilot`.
